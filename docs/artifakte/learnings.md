* Verifikation: mit anrufen an einer nummer und da von 2min 3x ist schon ungewohnt. Erste Besstellung ihrer Public Cloud Lösung wollte dann gerade Geld. zum glück wird neben credit karte auch twint angeboten, was natürlich ein + ist. Aber von einem Swiss Provider auch erwartbar ist. Leider wurde diese Order dann aber gespert da mein verifikation noch nicht abgeschlossen gewesen sei. Alles gut, aber dann wenn man diesen fortfahren möchte erschien nur ein Meldung, Fehler aufgetreten, ohne eine anschluss zu geben, was zu tuen ist. Darauf hin, habe ich am sontag ein Ticket eröffnet, da die telefon nummer nur von from 09h00 to 18h00 unter der woche für non premium users zur verfügung steht.
* MFA Prozess war gut markiert, "Action Required" finde ich positiv, das man den user einfach dafür enabled. und prozess lief sehr angenehm
* Der Zugang zu jleastic cloud war initial schwiergi zu finden, der name ist aus meiener sicht ungewohnt, und verglichen mit dem sehr eindeutigen portfolio von informaniak ungewohnt. der manager wirkt aber aufgeräumt und nicht überladen verglichen zu andern grossen providern. 
* Das jelastic cloud dashboard sieht unbekannt charmant aus, man sieht hier wurde einiges anders gedacht und in einen kleineren einfacher fokus als die big players, ui ist verschachtelt, und funktionen gibt es wenige, aber wenn man sie findet ist klar was sie machen. Die fehlermeldung waren aber sehr bedürfigt. Zum Beispiel: "The [jem vcs update] operation has failed: Project updated failed" hierzu musste ich dann auf die logs file zu greifen, die basicly einfach einen ftp wrapper sind und den zugriff möglich machen wenn man das technische verständnis hat, aber für Einsteiger sicher ungeschickt. Aber mit Azure und logs weiss ich auch das, grosse provider sich schwer tuen, dies angenehm dem User ersichtlich zu machen. 

* Git integration funktionierte wunderbar, viel möglichkeiten gibt es hier aber nicht, musste meine projekt struktur anpassen, da es nicht monorepo supported.
